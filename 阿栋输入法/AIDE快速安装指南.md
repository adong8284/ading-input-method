# AIDE快速安装指南

## 📱 步骤1：接收并解压文件
1. 在QQ中接收`阿栋输入法.zip`文件
2. 保存到手机存储（建议：`/storage/emulated/0/Download/`）
3. 使用文件管理器解压到：`/storage/emulated/0/AndroidStudioProjects/阿栋输入法`

## 🛠️ 步骤2：简化项目配置
如果遇到编译错误，请按以下步骤简化项目：

### 方法A：使用简化版本（推荐）
1. 重命名文件：
   ```
   AndroidManifest.xml → AndroidManifest-original.xml
   AndroidManifest-simple.xml → AndroidManifest.xml
   
   build.gradle → build-original.gradle
   build-simple.gradle → build.gradle
   ```

2. 删除复杂模块（可选）：
   - 删除`modules/`文件夹中的复杂文件
   - 只保留`SimpleInputMethodService.kt`和`SettingsActivity.kt`

### 方法B：逐步添加功能
1. 先编译基础版本
2. 逐步添加功能模块
3. 每次添加后重新编译测试

## ⚠️ 常见问题解决

### 问题1：Gradle同步失败
**解决方法**：
1. 检查网络连接
2. 在AIDE设置中配置代理（如果需要）
3. 重启AIDE重试
4. 清理缓存：AIDE菜单 → 清理项目

### 问题2：找不到设备
**解决方法**：
1. 确认USB调试已开启
   - 设置 → 关于手机 → 连续点击"版本号"7次
   - 返回设置 → 开发者选项 → 开启USB调试
2. 重新连接手机
3. 重启手机和AIDE

### 问题3：编译错误
**解决方法**：
1. 查看错误日志，找到具体错误
2. 降低targetSdkVersion到30
3. 移除有问题的依赖库
4. 使用简化版本（方法A）

### 问题4：内存不足
**解决方法**：
1. AIDE设置 → 增加Java堆内存到1024MB
2. 关闭其他应用
3. 重启手机

## 🚀 快速测试步骤

### 第一步：编译安装
1. 使用简化版本配置
2. 点击运行按钮
3. 等待安装完成

### 第二步：启用输入法
1. 打开手机设置
2. 进入"语言和输入法"
3. 找到"虚拟键盘"
4. 启用"阿栋输入法"

### 第三步：切换输入法
1. 在任何输入框长按
2. 选择"输入法"
3. 选择"阿栋输入法"

### 第四步：测试功能
1. 打开记事本或聊天应用
2. 测试字母输入
3. 测试数字输入
4. 测试空格和退格
5. 点击设置按钮进入设置页面

## 🔍 调试技巧

### 查看日志：
1. AIDE菜单 → 查看 → Logcat
2. 过滤标签：`AdingInputMethod`

### 断点调试：
1. 在代码行左侧点击设置断点
2. 以调试模式运行
3. 查看变量值

### 实时修改：
1. 修改代码后保存
2. AIDE会自动重新编译
3. 重新运行即可生效
